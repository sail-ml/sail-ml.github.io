Softmax¶

class sail.modules.Softmax¶

Applies the softmax function to the input data, which squashes the inputs to sum to 1 along dimension 1

\[\text{Softmax}(x_{i}) = \frac{\exp(x_i)}{\sum_j \exp(x_j)}\]

Note

Softmax is a subclass of Module. See Module to find the base functions which are implemented in Softmax.
