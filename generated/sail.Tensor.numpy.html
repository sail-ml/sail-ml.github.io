sail.Tensor.numpy¶

Tensor.numpy() → Numpy Array¶

Returns the numpy representation of the data

This functions returns a copy of the data as a numpy array.

Example

>>> x = sail.random.uniform(10, 20, (10, 20))
>>> x.shape
(10, 20)
